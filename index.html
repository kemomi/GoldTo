快速开始

三步完成部署，无需任何 API Key 即可完整体验。

1. 克隆与安装 bash
git clone https://github.com/kemomi/GoldTo.git
cd GoldTo
bash setup.sh  # 一键安装所有依赖
2. 配置（可选） .env
# 留空 → 自动 Mock 模式，完整功能无需 API Key
LLM_API_KEY=your_key_here  # 可选
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
ZEP_API_KEY=  # 可选，用于持久化记忆
3. 启动 bash
bash dev.sh
# 前端 → http://localhost:3000
# 后端 → http://localhost:5001 (API docs: /docs)
